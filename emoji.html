Introduction to Emoji package

Taikgun Song

2017-10-28

When working with online text data such as twitter messages, Emoji Unicodes are often included as part of the data. Emoji characters are rich with underlying symbols, meanings, and expressions, but have been neglected in traditional text mining analysis.

The emoji package is capable of the following:

  1. Provide basic information of Emoji character given its Unicode.
  • Name of the Emoji
  • Categorize Emoji to pre-determined dimension assigned by the Unicode Consortium.
  • Categorize Emoji to topic dimension determined by the author of this package.
  • Provide sentiment information based on the tidytext package.

  1. Plot image of the Emoji given its Unicode.

  2. Search Emoji based on keywords.

Data: emoji_v5.0

The data of the current package comes from Emoji List v5.0. The table from the web page is contained in data(emoji_v5.0). The data set consist of the following information:

  • Name of the Emoji
  • Unicode of the Emoji
  • Three hierarchical categories of Emoji. cat1 being the largest, cat 2 is a sub category of cat1, and so on.
  • Keyword of the Emoji
library(emoji)
data("emoji_v5.0")

Extract Emoji Information with emoji_info()

The intention of the emoji package is to reflect the information of the Emoji while conducting text analysis. The package minizie the information loss by providing names, keywords, and categories of Emoji. While the previously listed data are basic materials provided by the Unicode Consortium, the package goes a step further by providing sentiment of the given Emoji unicode using the get_sentiment() function from the tidytext package. Also, the categories assigned by the Unicode Consortium is very limited, thus additional categories are implemented by the author. Function, emoji_info(), is the tool to derive the mentioned information.

Arguments

  • unicode Unicode of an Emoji character.

  • lexicon_type type of lexicon to be used for sentiment and score. This feature is based on the tidytext package and the following three types are available: bing, nrc, and afinn. The default lexicon dictionary is bing. Learn more about tidytext package and get_sentiment() function at https://cran.r-project.org/web/packages/tidytext/tidytext.pdf.

Results

  • name Name of the Emoji assigned by the Unicode Consortium.

  • sentiment_and _score sentiment and score of the Emoji by the get_sentiment() function from the tidytext package.

  • categories Categories of the Emoji assigned by the Unicode Consortium.

  • categories2 Categories of the Emoji assigned by the author of this package.

Examples

View Emoji with plot_emoji()

When Emoji is collected as text data, it is often saved as a code; mainly in “UTF-8”, but could be in other encoding formats. Considering that Emoji a pictogram, one can cast an image of a Emoji character by using plot_emoji() function.

Arguments

  • unicode Unicode of an Emoji character.

Results

  • The image of the Emoji will be plotted along with its unicode and name.

Examples

emoji_plot("U+1F600")

Evaluate a single character string with Emoji via emoji_info_table()

Arguments

  • char_string A single character string that may contain zero or more Emoji character

  • lexicon_type type of lexicon to be used for sentiment and score. This feature is based on the tidytext package and the following three types are available: bing, nrc, and afinn. The default lexicon dictionary is bing. Learn more about tidytext package and get_sentiment() function at https://cran.r-project.org/web/packages/tidytext/tidytext.pdf.

Results

  • uni_code Unicode of the Emoji used in the string.

  • count Number of each Emoji used in the string.

  • name Name of each Emoji used in the string.

  • score sentiment or score of each Emoji used in the string.

  • categories Categories of each Emoji used in the string. The categories are assigned by the Unicode Consortium.

  • categories2 Similar to categories, but it was assigned by the author of the package.

Examples

a_string="Obviously #hateher  U+1f495  U+1f60f  U+1f469  @ Los Angeles, California https://www.instagram.com/p/BCbPSxxGVua/"
